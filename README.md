# 2026 목표 트래커

2026년 목표를 추적하는 웹 애플리케이션입니다.

## 실행 방법

```bash
# 개발 서버 실행
npm run dev

# 빌드
npm run build

# 빌드된 앱 미리보기
npm run preview
```

## 주요 기능

### 운동 관리 페이지
- **월별 캘린더 뷰**: 날짜를 클릭하여 운동한 날을 표시 (녹색)
- **연간 뷰**: 3x4 그리드로 12개월 전체 보기
- **진행률 바**: 월별 목표 대비 진행률 표시 (100% 초과 가능)
- **목표 설정**: 월별 목표 일수 설정 가능
- **데이터 저장**: localStorage를 통한 자동 저장

### 테마 시스템
5가지 테마 중 선택 가능:
- 밝은 테마 (기본)
- 어두운 테마
- 바다 테마
- 노을 테마
- 숲 테마

### 향후 추가될 페이지
- **홈**: 전체 대시보드
- **돈**: 지출 추적
- **도파민**: 습관 추적

## 기술 스택
- React 18
- React Router
- Vite
- CSS3 (애니메이션 포함)
